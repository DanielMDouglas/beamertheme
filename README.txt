This beamer template is based upon the Drexel IceCube beamer template by Mike Richman (michael.d.richman@drexel.edu)

To use this template, first download the latest version:

git clone https://github.com/neutrinope/beamertheme docfolder

where `docfolder` is a folder for the specific document you are about to write.

Then copy or rename example.tex to the desired name and fill it with your own content!
